# Tabor Synergy AI — Patient Prompt Guide
### What to say to your AI front desk (Aria)

> Share this with your patients so they can start using the AI front desk right away.
> Open the chat bubble and type any of the prompts below — exactly as written.

---

## 👋 Getting Started

```
Hello
```
```
Hi, what can you help me with?
```
```
I'm a new patient at this clinic.
```
