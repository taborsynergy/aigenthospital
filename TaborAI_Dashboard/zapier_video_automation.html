🔧 Setup Instructions (One-Time)

Step 1: Enable Google Cloud Text-to-Speech API
1. Go to console.cloud.google.com
2. Create new project (name: "Tabor Video")
3. Enable APIs: Search for "Text-to-Speech API" → Click "Enable"
4. Create service account:
  • Go to "Service Accounts"
  • Create new service account → Name: "zapier-tts"
  • Create JSON key → Download & save
5. Enable Google Drive API in same project
Step 2: Connect Google Cloud to Zapier
1. Go to zapier.com → Dashboard
2. Create new Zap: "Make a Zap"
3. Trigger: Google Forms or Webhook (to submit script)
  • Use Webhook for maximum flexibility
4. Test the trigger (you'll need sample data)
5. Add Action: "Google Cloud Text-to-Speech"
  • Authenticate with service account JSON key
  • Configure: Input text → Output audio
Step 3: Save Audio to Google Drive
1. Add another action: "Google Drive - Create File"
2. Folder: Create new folder "Tabor Video Assets"
3. Map output from TTS → save as MP3
  • Filename: {{Timestamp}} + "_voice.mp3"
4. Test action to verify it saves
Step 4: Send Notification When Ready
1. Add final action: "Gmail - Send Email"
2. To: your email
3. Subject: "Your Video Audio is Ready"
4. Body: Include link to MP3 file in Google Drive
5. Turn on Zap

⚙️ Complete Zap Configuration

Trigger: Webhook (Catch Raw Hook)

POST Webhook URL: https://hooks.zapier.com/hooks/catch/YOUR_ID/ Expected Input JSON: { "script": "Hi! I'm Dinakar from Tabor Synergy...", "voice": "en-US-Neural2-A", "email": "your@email.com" }

Setting Value
Input Text {{script}} (from webhook)
Language Code en-US
Voice Name en-US-Neural2-A (professional female)
Speaking Rate 1.0
Audio Encoding MP3

Folder Tabor Video Assets
Convert to Google Format No

🚀 Quick Start: Run the Automation

Once Zap is Set Up (takes 30 min first time)
To create a video:

1. Paste this into your browser console or use curl:
curl -X POST https://hooks.zapier.com/hooks/catch/YOUR_WEBHOOK_ID/ \ -H "Content-Type: application/json" \ -d '{ "script": "Hi! I am Dinakar from Tabor Synergy...", "voice": "en-US-Neural2-A", "email": "your@email.com" }'
2. Wait 2 minutes
3. Check email → Google Drive link appears
4. Download audio MP3
5. Create Canva slides (2 min)
6. Combine in CapCut (3 min)
7. Upload to Loom → Done!

OR Use Google Forms for Easier Submission

4. Rest is same automation

Benefit: Non-technical (just fill out form)

🔧 Troubleshooting

❓ "Zapier says 'Insufficient Permissions'"

→ Google Cloud service account needs Editor role
→ Or try: Generate new JSON key from Google Cloud

❓ "Audio file not saving to Google Drive"

→ Verify Google Drive API is enabled
→ Check folder permissions (allow Zapier access)
→ Try uploading manually to test

❓ "Audio sounds robotic"

→ Change voice: en-US-Neural2-C (more natural)
→ Reduce speaking rate to 0.9
→ Break script into shorter paragraphs

❓ "Webhook URL not working"

→ Use Google Forms trigger instead (simpler)
→ Or use Zapier's "Code" app to test webhook
